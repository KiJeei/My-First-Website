"Motherboard"

The backbone that ties the computer's components together at one spot and allows them to talk to each other. Without it, none of the computer pieces, such as the CPU, GPU, or hard drive, could interact.

"Keyboard"

An input device that allows you to type letters, numbers, and symbols into your computer or other electronic device.

"Hard Drive"

A physical data storage device that stores digital information. Hard drives are used to store both operating systems and user files, such as photos, music, documents, and videos.

"Mouse"

A small device that a computer user pushes across a desk surface in order to point to a place on a display screen and to select one or more actions to take from that position.

"Ram"

is a computer's short-term memory, where the data that the processor is currently using is stored.
